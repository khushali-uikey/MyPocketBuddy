MyPocketBuddy 💸

A simple and responsive personal expense tracker built with PHP and MySQL to help users manage their money, track spending, and maintain transaction history.

✨ Features

- 🔐 User Signup & Login Authentication
- 🔑 Secure Password Hashing
- 👤 Session Management
- 💰 Add and Manage Budget
- 📝 Add Transactions
- 📂 Expense Categories
- 📜 Transaction History
- 📊 Remaining Balance Calculation
- 🔄 Reset Password Option
- 📱 Fully Responsive Design

 🛠 Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Git & GitHub
- InfinityFree (Deployment)

⚙ How It Works

1. Create an account  
2. Login securely  
3. Add your budget  
4. Add expenses with category, amount, description, and date  
5. View your remaining balance  
6. Track all transactions in history  

🗄 Database Structure

 Users Table
Stores user details:
- id
- name
- email
- password

Budget Table
Stores user budget:
- id
- user_id
- total_amount

 Transactions Table
Stores user transactions:
- id
- user_id
- amount
- category
- description
- date
